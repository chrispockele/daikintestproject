DEVELOPED USING: 
 * GCC c++ version 8.3.0
 * cmake version 3.13.4
 * Google Test version 1.8.1

Tested on a Debian 10 installation with all of the above installed using Debian package management.


HOW TO BUILD AND RUN
********************
In the command line:
 - change to the "build" directory;
 - run the following command: "cmake ../src"
 - run the following command: "cmake --build ."

Result is a binary called "DaikinGTestRun", intended to be called from the command line.
