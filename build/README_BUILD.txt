DEVELOPED USING: 
 * GCC c++ version 8.3.0
 * cmake version 3.13.4


HOW TO BUILD AND RUN
********************
In the command line:
 - change to the "build" directory;
 - run the following command: "cmake ../src"
 - run the following command: "cmake --build ."

Result is a binary called "DaikinTestRun", intended to be called from the command line.
